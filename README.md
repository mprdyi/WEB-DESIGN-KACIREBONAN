# WEB-DESIGN-KACIREBONAN
# 🌊 WEB-KACIREBONAN
### Digital Showcase of Cirebon Cultural Heritage

WEB-KACIREBONAN adalah website eksplorasi budaya yang dibangun sepenuhnya menggunakan **Pure Native HTML, CSS, dan JavaScript (Vanilla)** tanpa framework maupun library tambahan.

Website ini menghadirkan 33 halaman interaktif yang menampilkan kekayaan budaya Cirebon secara visual, informatif, dan edukatif.

---

# 🧠 Filosofi Pengembangan

Proyek ini dibangun dengan pendekatan:

✔ Tanpa Framework  
✔ Tanpa Library  
✔ Tanpa Template  
✔ Full Manual Coding  
✔ Struktur Modular Native  

Tujuan utama:
- Melatih fundamental web development murni
- Mengoptimalkan performa tanpa dependensi eksternal
- Menunjukkan kemampuan rekayasa struktur front-end secara manual
